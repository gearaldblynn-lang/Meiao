# Local Work Tracker

This directory stores local PRDs, task breakdowns, and investigation notes for agent work.

GitHub is used for version storage and backup in this project; local Markdown is the default issue tracker unless the user explicitly asks to create GitHub issues.

Recommended structure:

```text
.scratch/<feature-slug>/PRD.md
.scratch/<feature-slug>/issues/01-short-title.md
.scratch/<feature-slug>/issues/02-short-title.md
```
