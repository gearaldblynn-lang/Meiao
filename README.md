# Meiao

梅奥视频分镜生成器（前端原型）。

## 运行

```bash
python3 -m http.server 4173
```

访问 `http://localhost:4173`。

## 关键能力

- 苹果极简 UI，左侧配置可收起，右侧项目工作区可扩展。
- API Key 移入“设置”弹窗，不在主配置区明文展示。
- 支持脚本逻辑预设（高转化电商短视频）与自定义逻辑。
- 新增国家/语言配置（默认 中国/中文），口播语言跟随该配置。
- 上传产品图支持 JPEG/PNG/WEBP、单张≤30MB、最多10张。
- 先将产品图上传到 Kie File Upload（file-stream-upload）得到 URL。
- 将全部配置发给 Gemini（gemini-3-flash）生成：
  - 分镜图 prompt（含连续性规则）
  - 完整分镜脚本文本（禁止字幕格式）
- 按分镜 prompt + 视频比例 + 产品图 URL 调用 nano-banana-2 生图。
- 工作区支持：
  - 分镜 prompt 摘录
  - 分镜生图按钮 / 进度显示
  - 超时后基于 taskId 重试轮询
  - 图片放大预览与左右切换
  - 完成后自动拼合 3/6/9 宫格，30 秒为前6+后6 两个6宫格
  - 打包下载（拼合图+单图+脚本文本）
