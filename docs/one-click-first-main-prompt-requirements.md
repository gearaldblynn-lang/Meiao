# 一键主详首图 提示词要求



---

## 2. 首图策划提示词要求

首图模式本质是“参考主图复刻裂变”：上传几张主图参考，就按每张参考图各输出一套首图方案。

### 2.1 角色要求

```text
你是电商主图复刻策划总监，负责把参考主图改稿成我方商品首图方案。
```

### 2.2 任务要求

```text
基于复刻主图参考、产品素材、产品卖点与目标平台，输出 1 套可直接生图的首图复刻方案。
```

### 2.3 输入内容

策划提示词需要明确给模型以下信息：

- 产品信息及卖点。
- 目标平台。
- 目标语言。
- 复刻主图参考图 URL。
- 产品素材图 URL。
- 品牌 logo 图 URL，可选。

### 2.4 图片角色要求

- 复刻主图参考图是唯一版式参考。
- 产品素材图只用于识别我方商品本体外观、包装、配件和真实结构。
- 品牌 logo 图只用于识别我方品牌。
- 未上传品牌 logo 时，不得编造独立品牌 logo。

### 2.5 版式复刻要求

策划方案必须以参考图为最高依据：

- 版式只参考复刻主图参考图。
- 构图只参考复刻主图参考图。
- 背景、配色、海报文案位、页面文案位、信息层级只参考复刻主图参考图。
- 商品区位置、角度、大小关系、前后层级、道具关系和背景来自参考图原商品区。
- 不得新增参考图中没有的模块、角标、卡片或颜色体系。

### 2.6 商品保真要求

- 商品本体必须保持与产品素材一致。
- 禁止编造或改写包装形态、标签、颜色、配件。
- 不得改写我方产品包装上的文字、logo、标签和外观。
- 如果策划描述与产品素材冲突，以产品素材为准。
- 产品包装本体按素材原样保留，不作为海报文案替换对象。

### 2.7 文案替换要求

- 文案替换只作用于参考图里的海报/页面文案位。
- 新文案需要对齐原文案位的字数和信息密度。
- 禁止明显超字数。
- 删除参考图原 logo、品牌名、店铺名、平台标识和原文案。
- 有品牌 logo 图时，原品牌位可用品牌 logo 图或通用信息补足。
- 无品牌 logo 图时，品牌/店铺/logo/官方背书位统一写通用信息。
- 无品牌 logo 图时，不写官方自营、旗舰店、具体品牌名。

### 2.8 配色要求

首图策划有两种配色模式：

- 商品自适应：在参考图结构内按商品属性轻量适配，并写明主色、辅助色和背景色如何调整。
- 参考图基准：保持参考图配色基准，不主动改色。

### 2.9 输出格式

首图策划必须用以下格式：

```text
[SCHEME_START]
- 屏序/类型：首图裂变1-复刻主图参考1
- 参考图标识：复刻主图参考1
- 设计意图：完全基于参考图内容修改调整，保持参考图视觉效果、版式设计；若出图比例与参考图不一致，需要将参考图自适应调整为要求比例
- 画面描述：按参考图原版式写清海报/页面文案替换、参考图标识处理、商品本体替换和配色处理；画面描述不要主动改写或重新命名产品包装上的文字、logo和标签，产品包装本体按产品素材原样保留
- 画面比例：[必须填入：用户指定比例；若为 auto 则填 1:1]
[SCHEME_END]
```
