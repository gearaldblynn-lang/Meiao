# 服务端收口与任务队列验收清单

这份清单用于确认两件事是否真的可交付：

1. 外部 API Key 和模型调用已从前端收回服务端
2. 统一任务队列、重试、取消、恢复、排障信息已在真实流程中可用

## 建议验收顺序

1. 基础环境
2. 通用任务队列
3. 逐模块业务
4. 管理与排障

## 基础环境验收

- 打开 `http://localhost:3000`，确认本地开发页可访问
- 打开 `http://127.0.0.1:3100/api/health`，确认后端健康检查返回正常
- 执行 `npm run doctor`，确认 3000、3100 和代理检查状态
- MySQL 模式下确认 `internal_jobs` 表已创建
- 设置页确认只显示只读系统状态，不显示真实密钥
- 浏览器 Network 面板确认不再出现直连：
  - `api.kie.ai`
  - `ark.cn-beijing.volces.com`
  - `runninghub.cn`
  - `kieai.redpandaai.co`
- 浏览器请求头里确认不再出现第三方 Bearer Key

## 通用任务队列验收

- 任意模块提交任务后，确认内部任务状态按下面流转：
  - `queued`
  - `running`
  - `succeeded / failed / cancelled`
- 提交多个任务，确认超过并发上限时会排队
- 制造瞬时错误，确认进入重试等待
- 制造鉴权错误、参数错误、任务不存在，确认直接失败
- 任务运行中刷新页面，确认状态和结果仍能恢复
- 找回功能确认走内部恢复接口，不是前端直查第三方
- 取消任务后：
  - 排队中的任务直接取消
  - 运行中的任务停止推进或标记取消
- 重试任务后，确认日志中能看到重试痕迹

## 逐模块业务验收

### 一键主图 / 一键详情
- 策划可用
- 生成可用
- 找回可用
- 重试可用
- 中断可用
- 完成结果能正常回写页面状态

### 出海翻译
- 上传可用
- 处理可用
- 结果展示可用
- 刷新页面后记录仍在

### 产品精修
- 分析任务可用
- 生成任务可用
- 单任务找回可用
- 重试可用
- 中断可用

### 买家秀
- 策划可用
- 图片生成可用
- 结果恢复可用

### 短视频
- 长视频主流程可用
- Veo 流程可用
- 分镜脚本可用
- 分镜板可用
- 白底图可用
- 刷新页面后视频结果仍能恢复

### 素材上传
- 上传成功后由内部接口返回素材地址
- 同一用户素材路径保持隔离

## 管理与排障验收

- 管理员日志页可按功能、人员、结果筛选
- 日志中至少能看到：
  - 内部任务 ID
  - 外部任务 ID
  - provider
  - 重试次数
  - 错误摘要
- 员工账号不能看到真实密钥
- 员工账号不能修改系统级敏感配置
- 环境变量缺失时，服务端返回明确报错
- 本地模式访问任务队列接口时，返回明确提示“不支持完整内部任务队列”
