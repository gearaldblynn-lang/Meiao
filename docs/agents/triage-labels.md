# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the label strings used in this repo's local Markdown issue tracker.

| Label in skills | Label in this repo | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter or missing environment/log details |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an agent to implement or investigate |
| `ready-for-human` | `ready-for-human` | Needs human product/design/ops decision before implementation |
| `wontfix` | `wontfix` | Will not be actioned |

Record the status near the top of each local issue:

```markdown
Status: ready-for-agent
```
